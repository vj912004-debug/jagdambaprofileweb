# Shree Jagdamba Steel Profile Website

Next.js website for Shree Jagdamba Steel Profile — MS & SS plate stocking, laser cutting, CNC profile cutting, and pan-India delivery.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy

Deploy with [Vercel](https://vercel.com) or any Node.js hosting that supports Next.js.
